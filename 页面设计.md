##### 师资力量

![师资力量](页面设计.assets/1556801852408.png)

**19.5.3完成**

这里遇到一个问题，头像不能居中，找了好多的方案都是失败的。使用了wordpress中的一个方法就成功了。这里记录下来：

```css
/* Alignment */
.alignleft {
 display: inline;
 float: left;
}
.alignright {
 display: inline;
 float: right;
}
.aligncenter {
 clear: both;
 display: block;
 margin:auto;
}
```

最后发现，Bootstrap中就有样式，thumbnail缩略图可以实现这个功能，所以直接换成这个样式更好用。然后添加了模态框，modal.js实现了点击弹窗的功能。
